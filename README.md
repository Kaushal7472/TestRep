# IBM Data Analyst Capstone — IT Skills Demand Analysis

Analysed Stack Overflow developer survey data to identify current technology 
usage trends and future skill demand across the global developer community.

**Tools:** Python · Pandas · NumPy · Matplotlib · Seaborn · IBM Cognos Analytics

**Key steps:** API data collection · Web scraping · Data wrangling · EDA · 
Outlier detection · Dashboard development
